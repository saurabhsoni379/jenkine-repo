Manage AWS Default VPC

View Default VPC

To identify the default VPC in your AWS account, you can use the AWS Management Console or AWS CLI.

AWS Management Console

  1. Open the VPC console at AWS VPC Console.
  2. Click on "Your VPCs" in the navigation pane.
  3. Look for the column labeled "Default VPC". The default VPC will have "Yes" in this column.

AWS CLI

aws ec2 describe-vpcs --query "Vpcs[?IsDefault==true].VpcId"

Create Default VPC

If you need to create a default VPC, follow these steps:

AWS Management Console

  1. Open the VPC console at AWS VPC Console.
  2. Click on "Your VPCs" in the navigation pane.
  3. Click on the "Actions" button.
  4. Select "Create Default VPC" from the dropdown menu.
  5. Confirm the action when prompted.

AWS CLI

aws ec2 create-default-vpc

Verify Default VPC

After creating the default VPC, you can verify its creation:

AWS CLI

aws ec2 describe-vpcs --query "Vpcs[?IsDefault==true].VpcId"
